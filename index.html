The format

Purposeful, not faster.

Make It So Camp teaches people to say what they mean clearly enough that it can be carried out, reviewed, and shared. Participants bring real work, make the hidden method visible, write it down, and use it to build a rough working thing.

01

Welcome + Frame

The positioning shift: purposeful, not faster.

Start from a real ambition rather than a task list. Name what you want to build, not only what you want to accelerate.

02

Shift 2 · Methods You Can't See

The articulation shift: the bottleneck is articulation, not the tech.

Translate tacit judgment into instructions, constraints, and decision logic that another actor can follow.

03

Shift 3 · Write It Down, It Compounds

The reuse shift: documented once, reused all day.

Turn the method into a head-start document so the next build begins from shared context instead of a blank page.

04

Shift 4 · Explicit = Shareable

The collaboration shift: if it is explicit, someone else can run it.

Compare methods with peers, find overlaps, and leave with a protocol that can be challenged and improved.

Camp instances

Two rooms, one method.

Each camp is local to its host, but the structure is shared: before-camp interviews, a day of articulation exercises, and a second day of building and demos.
